# 我的刷题步骤

## `1.新建刷题目录`
比如第 `1` 题 two-sum

`go run main.go -a=1`

比如第 `100` 题 same-tree

`go run main.go -a=100`

## `2.模板流`
个人建议刷完前50题后，完全刷一个系列的，比如我最近在刷 Tree，因为 Tree 是最简单的～

刷题过程中要整理好自己的解题模板

切记！！！！不要相信 `自己解题可以提高智商` 这种鬼话， 5分钟没思路直接看答案，千万别浪费时间

然后 `模板流` 一把嗖，目标是 `所有算法全是手！速！题！`

## `3.目录约定`  `生成 readme.md`

`分类` : `1 级目录`

`题名` : `2 级目录`

`算法` : `3-N 级目录`

执行 `go run main.go` 自动生成 `readme.md`
